# open-webui-code-aware-context-manager
Filter to manage context in a more efficient way for big amounts of code context.

# features
- Capable of following the state of your code, propose commits, apply commits, the state of such commits (already applied, or not).
- Persistent meta cognitive awareness. If needed, a new conversation can retrieve high level context for a database.
- Esencially, the LLM will know what you have been working on lately, its current state, and how to continue.
